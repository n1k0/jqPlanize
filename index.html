jQPlanize plugin for jQuery

The planize jQuery plugin adds some hierarchical data organization features to given DOM node containing HTML headings:

  • adds numerotation and anchors in front of all headers
  • generates an HTML table of content
  • degrades gracefully if javascript is not available/enabled

See it in action!

The examples below will run on the current page HTML source code. At any time you can .

Enumeration starting from H1

$('html *').planize();

Enumeration starting from H2

$('html *').planize({ min_level: 2 });

Enumeration starting from H2, with a custom number suffix

$('html *').planize({ min_level: 2, number_suffix: ')' });

Enumeration starting from H2 headings and Table of Contents generation

$('html *').planize({ min_level: 2, generate_toc: true, toc_elem: $('#toc') });

Enumeration starting from H2 to H3 headings, and Table of Contents generation

$('html *').planize({ min_level: 2, max_level: 3, generate_toc: true, toc_elem: $('#toc') });
